# Nathalies Tortenwelt

Eine wunderschöne One-Pager Website für eine Tortenbäckerei.

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment auf Vercel

1. Push zu GitHub
2. Verbinde Repository mit Vercel
3. Deploy!

Oder nutze die Vercel CLI:

```bash
npm i -g vercel
vercel
```
